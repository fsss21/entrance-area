# Данные по страницам

| Файл | Страница | Описание |
|------|----------|----------|
| `scientific-activity.json` | /scientific-activity | Кнопки (исследования, публикации и т.д.), список мероприятий с датой/временем |
| `educational-programs.json` | /educational-programs | Кнопки (экскурсии, лекции, мастер-классы), список программ с датой и названием |
| `collection.json` | /collection | Список названий коллекций для карусели |
| `service-building.json` | /service-building | Данные по этажам 1 и 2: заголовок, кнопки залов, текст плана |
| `main-house.json` | /main-house | Кнопки залов (9–15), текст плана главного дома |
| `actual-info.json` | /actual-info/work, /actual-info/time, /actual-info/meetings | branches: для каждой кнопки (уткина дача и др.) — workInfo (режим работы, адрес, телефон, e-mail), practicalInfo (блоки для модалки «практическая информация»); scheduleContent — расписание и текст для секции «мероприятия» |
